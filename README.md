## Instructions

- Settings in the screenshot include:
```json
{
  "editor.renderLineHighlight": "all",
  "editor.renderWhitespace": "all",
  "editor.cursorStyle": "block",
  "editor.fontFamily": "'Cascadia Code PL', 'monospace', monospace",
  "editor.fontLigatures": "'calt', 'ss01'",
  "editor.rulers": [
    100,
    120
  ]
}
```
